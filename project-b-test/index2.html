Melody Maker

INSTRUCTION Mouse Click Any Note To Activate/Mute It Keyboard Press Space to Start
DESCRIPTION This is a Melody Maker. Press any rectangle, and it will light up to show it is going to be played. Click again to undo. After you finished your melody, press space to see how the complete version would look like. When it is playing, press spcae again so that you can stop and edit.
